# Project Overview

This application is a **multi-page analytics dashboard** built using **Streamlit**.

Streamlit is an open-source Python framework that allows developers and data scientists to build interactive web applications using only Python.

The dashboard demonstrates:

- Data analytics dashboards
- Machine learning deployment
- Interactive visualization
- User-friendly UI

---

# Machine Learning Model

The project includes an **Iris Flower Classification Model**.

The Iris dataset is one of the most famous machine learning datasets and contains **three flower species**:

- Setosa
- Versicolor
- Virginica

Each flower is classified using four features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The model predicts the species based on these measurements.

# Installation

### Clone the repository

```
git clone https://github.com/Tmmfaris/streamlit-data-analytics-dashboard.git
```

### Navigate to the project folder

```
cd streamlit-dashboard
```

### Install dependencies

```
pip install -r requirements.txt
```

### Run the application

```
streamlit run app.py
```
